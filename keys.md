# All key combinations

## Combinations with dead keys

| base | char | name     |
| ---- | ---- | -------- |
| _letter_  | à | letter with grave accent |

| ---- | ---- | -------- |
| <kbd>,</kbd>  | ‚ | single low-9 quotation mark |
| _letter_  | á | letter with acute accent |
